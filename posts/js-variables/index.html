JS for "Chaynicks": Variables

Jan. 28, 2025

Types of variables:

We can declare variables in JS using these 3 keywords: var, let and const.

var Keyword

This keyword has a function-scoped or globally-scoped behaviour:

var y = 5;
var y = 15;

console.log(y)
console.log(y)
Output:
5
15

let Keyword

The let keyword has block scope and cannot be re-declared in the same scope:

let y = 17;
y = 42;

console.log(y)
Output:
4

Here we can update the value of variable: 

y = 42;

If we redeclare it by using let keyword again we will get an error that this variable already has been declared:

let y = 17;
let y = 42;

console.log(y)
Output:
SyntaxError: Identifier 'y' has already been declared

const Keyword

The const keyword declares variables that cannot be reassigned and it’s also block-scoped:

const y = 17;
console.log(y)
Output:
17

And if we’ll try to reassign the value:

const y = 17;
y = 20;
console.log(y)

We will get an error:

Output:
Type Error: Assignment to constant variable.
